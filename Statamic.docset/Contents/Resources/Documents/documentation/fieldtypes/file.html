File

The file fieldtype allows you to upload files through the control panel. Two parameters exist, allowed - an array of allowed filetypes, and destination - the directory (from root) you wish to save the file(s). File works exceedingly well in conjunction with the grid fieldtype.

header_image:
  display: Header Image
  type: file
  allowed: [png, gif, jpg, jpeg]
  destination: _assets/img/blog   # make sure your folder is writable
