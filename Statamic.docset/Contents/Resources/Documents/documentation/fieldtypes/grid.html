Grid

The grid fieldtype is a “meta” fieldtype, essentially a fieldtype full of fieldtypes. You can add as many columns as you desire of any existing fieldtypes, including the grid itself. We like to call these Inception Grids.

The configuration settings are identical to the fields.yaml itself, nesting a fields key under the grid field. The Grid also has a few options of its own, and also respects a width option for any of the fields inside it.

the_grid:
  display: The Grid
  type: grid
  starting_rows: 2
  min_rows: 2
  max_rows: 5
  fields:
    caption:
      display: Caption
      type: text
      width: 25%
    some_text:
      display: Some Text
      type: textarea
      width: 50%
    elk_or_moose:
      display: Elk or Moose
      type: select
      default: Elk
      width: 25%
      options:
        - Elk
        - Moose
        - Other

the_grid:
  display: The Grid
  type: grid
