Markitup

Markitup is a simple Markdown editor with a formatting toolbar to make life just a little easier. This is the default editor for your content field.

content:
  type: markitup
  display: Mark It Up!

note The only field that gets auto “markup”ed using this fieldtype is {{ content }}. Any YAML fields will need to use {{ your_field|markdown }} to correctly output markdown.
