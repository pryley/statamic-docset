Radio

The radio fieldtype acts very much like (and respects the same options as) the select fieldtype, but displays radio buttons instead of a select box.

dinner:
  display: Dinner Choices
  type: radio
  options:
    - Beans
    - Fish
    - Tree Bark
    - Elk
