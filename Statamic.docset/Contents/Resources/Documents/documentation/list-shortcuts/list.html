_list

Create a comma-separated list of values.

The _list shortcut will turn a simple YAML list into a comma-separated list.

Sample Usage

Input

tools:
  - hammer
  - pick axe
  - mallet
  - pocket knife

Tag

{{ tools_list }}

Displayed Result

hammer, pick axe, mallet, pocket knife

List Shortcuts

List shortcuts allow you to display variable lists into common structures for quick display.

  • _list
  • _spaced_list
  • _option_list
  • _ordered_list
  • _unordered_list
  • _sentence_list
  • _ampersand_sentence_list
  • _url_list
  • _spaced_url_list
  • _ordered_url_list
  • _unordered_url_list
  • _sentence_url_list
  • _ampersand_sentence_url_list
