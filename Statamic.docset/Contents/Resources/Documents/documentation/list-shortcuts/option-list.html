_option_list

Create an pipe-separated list of values.

The _option_list shortcut will turn a simple YAML list into a pipe-separated list.

Sample Usage

Input

categories:
  - technology
  - zoology
  - survival skills

Tag

{{ categories_option_list }}

Displayed Result

technology|zoology|survival skills
