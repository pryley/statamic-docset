_ordered_list

Create an HTML ordered-list of values.

The _ordered_list shortcut will turn a simple YAML list into an HTML ordered list.

Sample Usage

Input

tools:
  - hammer
  - pick axe
  - mallet
  - pocket knife

Tag

{{ tools_ordered_list }}

Displayed Result

<ol>
    <li>hammer</li>
    <li>pick axe</li>
    <li>mallet</li>
    <li>pocket knife</li>
</ol>
