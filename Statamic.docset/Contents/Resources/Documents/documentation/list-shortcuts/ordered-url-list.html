_ordered_url_list

Create an HTML ordered-list of taxonomy links.

The _ordered_url_list shortcut will turn a taxonomy list into an HTML ordered list of linked values. This URLs will be contextualized to the folder currently being viewed.

Sample Usage

Input

categories:
  - technology
  - zoology

Tag

{{ categories_ordered_url_list }}

Displayed Result

Assuming we’re viewing this tag on a page within the blog folder:

<ol>
    <li><a href="/blog/categories/technology">technology</a></li>
    <li><a href="/blog/categories/zoology">zoology</a></li>
</ol>
