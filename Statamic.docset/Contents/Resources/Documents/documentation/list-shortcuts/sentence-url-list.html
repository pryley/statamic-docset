_sentence_url_list

Create a sentence-like list of taxonomy links.

The _sentence_list shortcut will turn a taxonomy list into a sentence-like list of linked values.

If the list contains one item, this will return just that item. If the list contains two items, this will return both items with “and” between them. If the list contains three or more items, this will return a comma-separated list with “and” before the last item.

Sample Usage

Input

categories:
  - technology
  - zoology
  - survival skills

Tag

{{ categories_sentence_url_list }}

Displayed Result

<a href="/blog/categories/technology">technology</a>,
<a href="/blog/categories/zoology">zoology</a>, and
<a href="/blog/categories/survival-skills">survival skills</a>
