get

Retrieve GET variables.

{{ get:[variable] }}

Displays the value of the GET variable matching [variable].

Sample Usage

<p>
    You searched for `{{ get:search }}`.
</p>

Displays the content of the search GET variable.

Parameters

There are no parameters for this tag.
