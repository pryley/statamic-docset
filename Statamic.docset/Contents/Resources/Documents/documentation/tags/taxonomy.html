taxonomy

Display a list of taxonomy values.

{{ taxonomy:listing }}

Shows a listing of taxonomy values that have been used for the given parameters.

Sample Usage

<h2>Categories</h2>

<ul>
    {{ taxonomy:listing }}
        <li><a href="{{ url }}">{{ name }}</a></li>     
    {{ /taxonomy:listing }}
</ul>

Parameters

This tag accepts the following parameters:

  • type — required — the taxonomy field to look for
  • folder — the folder (or folders) to look through to find taxonomy values, defaults to the current URL; as of v1.5.3: you can use a * character at the end of your folder to select any entry whose folder starts with the text before the *, this will let you look in all subfolders easily
  • min_count — the minimum number of matching entries a value must have to show up in the list
  • sort_by — the field to sort the taxonomy list on, name by default
  • sort_dir — the direction the sorting should happen, either asc or desc, asc by default
  • limit — the maximum number of taxonomy values to display, null (no limit) by default

Available Tags

  • name — the taxonomy value
  • url — URL to view the taxonomy page for this value, based on the folder given
  • slug — the slug specifying this taxonomy’s folder and value
  • count — an iterator for the number of values displayed
  • first — a boolean as to whether or not the current loop is the first pass through or not
  • last — a boolean as to whether or not the current loop is the last pass through or not
  • results — the total number of content pieces found matching this taxonomy and folder
  • total_results — the total number of taxonomy values found with the given parameters
