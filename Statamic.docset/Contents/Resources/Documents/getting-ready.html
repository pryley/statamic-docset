Choosing a License

When to use which license.

Statamic has two licenses available for purchase: personal and professional. Buying either of these licenses will give you access to the same full-version of Statamic, the only difference being the level of the license key generated.

Which One Do I Need?

The type of license that you’ll need depends on how you plan to use Statamic.

Professional License (You probaby need this one)

A professional license is intended to be used in all standard and commercial scenarios. If the site is going to be used in anyway that’s related to people trying to make money, you’ll want this license. Some examples:

  • A company’s site (either for-profit or not-for-profit)
  • A start-up’s blog
  • A site for a church
  • A résumé/portfolio site (if you’re your own business)

Personal License

A personal license is intended for use in a person, non-commercia situation. Some examples of where a personal license is okay:

  • A personal blog (assuming you don’t make or intend to make a living off of it)
  • A school project
  • A family newsletter
  • A hobby site
  • A résumé/portfolio site (if you’re applying for jobs)

I’m Still Not Sure

If the above examples didn’t help you, just ask! We want you to know that the money made from license sales not only keeps our lights on, it also pays the team that builds Statamic, handles support requests, does all the heavy-lifting to make Statamic great, and funds our jerky addiction.

Upgrading or Transferring a License

Licenses cannot be upgraded or transferred to another person or company at this time.

Server Requirements

What your server needs to run Statamic.

It doesn’t take much to run Statamic on your server. Your server will need the following things:

  • PHP 5.3+, compiled with the mcrypt extension
  • mod_rewrite (or your server’s equivalent)

Checking Your Server

Can your server handle Statamic?

We’ve created a server check file that will determine if your server is capable of running Statamic. To use the check file:

  1. Download the free check file from the GitHub repository
  2. Upload check.php to your server
  3. Visit check.php in your browser

You should see the result of the check, including anything that’s missing. In some circumstances, the check file might say that the status of something is unknown. Know that here, “unknown” truly means unknown. It means that the check file wasn’t able to get a definitive answer about a particular feature, but your server may still support it (and Statamic may still work).

Acquiring Statamic

How to get a copy of Statamic.

To get a copy of Statamic, head over to the official Statamic store and buy the appropriate license for what you’re going to build. Once that’s done, you’ll be redirected to your Statamic account. Here you can see all of your past license purchases as well as download links for getting the latest version.

Use the Download link next to the license that you wish to use. This will download Statamic as a zip file onto your computer.
